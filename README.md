# CodeRabbit Practice Project

This is a sample Python project for testing CodeRabbit integration with Claude Code.

## Files

- `user_manager.py` - Simple user management system with CRUD operations
- `data_processor.py` - Data processing utilities

## Usage

```bash
python user_manager.py
```

## Purpose

This project contains intentionally simple code with room for improvements to test CodeRabbit's code review capabilities.
